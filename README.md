# basic_titanic_analysis
First practice-code to understand Pandas, my goal is to be able to analyse data efficiently.

##  Tools & Technologies
- **Python**: Core programming language.
- **Pandas**: Data cleaning and manipulation.
- **Matplotlib & Seaborn**: Data visualization.
- **Google Colab**: Interactive development environment.

##  Key Insights
- **Gender Factor**: Female passengers had a significantly higher survival rate than males.
- **Socio-Economic Factor**: Passengers in 1st class had a much higher probability of survival compared to other classes.
- **While Processing Data**: Handled missing values in the 'Age' column using mean imputation and removed the 'Cabin' column due to high sparsity.

##  How to Run
1. Open the `.ipynb` file in Google Colab.
2. Upload the `titanic.csv` file.
3. Run all cells.
